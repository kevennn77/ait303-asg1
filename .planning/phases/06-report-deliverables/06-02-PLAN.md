  <action>

    - In Microsoft Word: File → Export → Create PDF/XPS
    - Or: File → Save As → PDF
    - Verify: `report.pdf` opens correctly, all charts render, tables look correct, page count 10-15

    **Step 2 — Upload to cloud storage (per D-40)**
    
    *Primary — OneDrive:*
    - Create a folder: `AIT303_ASG1_Submission/`
    - Upload the entire `cloud-upload/` folder into it
    - Also upload `report.pdf` (the final submission PDF)

    *Secondary — Google Drive:*
    - Create a folder: `AIT303_ASG1_Submission/`

    - On OneDrive or Google Drive:

      - Set permissions: **View-only** (no edit access)
      - Copy the shareable link

    **Step 4 — Insert link into report (per D-43, DLVR-04)**
    - Open `report.docx` in Word

      > "All deliverables (notebooks, models, CSVs, charts) are available at: [INSERT CLOUD STORAGE LINK HERE]"
    - Alternatively, add a footnote on the title page or in the References section
    - Re-export to PDF: `report.pdf`

    **Step 5 — Submit to Moodle (per D-43, DLVR-02)**

    - Click Submit
