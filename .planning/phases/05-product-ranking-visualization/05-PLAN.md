## Input Data Contract

Phase 5 loads two data sources:

### 1. Phase 4 CSV — `aspect_labeled_reviews.csv` (D-35 schema)
| Column | Type | Description |
|--------|------|-------------|
| review_id | int | Sequential review ID |
| product_name | str | Product display name |
| review_text | str | Cleaned review text |
| aspect_label | str | CorEx-predicted aspect (Design, Sound Quality, Battery, Price, Build Quality, Features, Connectivity, Comfort/Portability) |
| aspect_confidence | float | CorEx max probability for assigned aspect |
| review_rating | float | Original Best Buy star rating (1.0-5.0) |
| review_date | str | Review date string |

### 2. IMDB Preprocessed Data — `preprocessed_imdb.csv`
Generated by Phase 1. Contains the `lemmatized` text column used to train the best sentiment model.

## Sentiment Model — Best Variant from Phase 2

**Winner:** SVM + TfidfVectorizer + Lemmatized
- Vectorizer: `TfidfVectorizer(ngram_range=(1,2))`
- Classifier: `LinearSVC(C=1.0, max_iter=5000, random_state=42)` (LinearSVC preferred over SVC(kernel='linear') for speed)
- Training data: IMDB 50K, 80/20 stratified split
- Reported metrics: Test F1 = 0.9091, CV Mean F1 = 0.9027

Phase 5 trains this model inline on the IMDB data (reproduces Phase 2 best variant) to keep the notebook self-contained.

## Aspect Definitions (8 aspects from Phase 4 CorEx)
Design, Sound Quality, Battery, Price, Build Quality, Features, Connectivity, Comfort/Portability
